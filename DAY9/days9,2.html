    • anchor tag

  • Basic Structure: An anchor tag is written as . It has an opening tag and a closing tag . The content that goes between these tags becomes the clickable part of the link.
  • Href Attribute: The most important attribute of an anchor tag is href, which stands for "hypertext reference". It specifies the URL of the page the link goes to.
  • Link Text: The text between the opening and closing tags is what the user sees and clicks on. This is called the link text or anchor text.
  • Target Attribute: The target attribute specifies where to open the linked document. It can take values like _blank (opens the linked document in a new window or tab) or _self (opens the linked document in the same frame or tab).
